## Installing

```bash
$ npm install meshblu-connector-rolling-spider
```

### Usage

```bash
$ npm start
```

or with debug

```bash
$ env DEBUG='meshblu-connector-rolling-spider*' npm start
```

### Releasing

```bash
$ npm run package
```
